###### Empty Comment

```
ctrl + ?
```

###### New line from where you are in the previous one

```
ctrl + Enter
```

###### Set the cursor to the beginning of all occurences of the same String

```
ctrl + D
```

###### Multiline comment

```
ctrl+K ctrl+C
```

###### Multiline Uncomment

```
ctrl+K ctrl+U
```

###### Copy the whole line down

```
Shift + Alt + Down Arrow
```
